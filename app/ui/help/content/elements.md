# Elementos de Asteroid

Asteroid es un editor visual que permite trabajar con diferentes tipos de elementos:

## Nodos
- **Nodo Base**: Elemento fundamental que contiene propiedades configurables
- **Nodo de Entrada**: Define puntos de entrada de datos
- **Nodo de Salida**: Define puntos de salida de datos

## Conexiones
- **Conexión Directa**: Enlace simple entre nodos
- **Conexión Múltiple**: Un nodo puede conectarse a varios destinos
- **Conexión Condicional**: Flujo condicional basado en propiedades

## Propiedades
Cada elemento en Asteroid tiene propiedades configurables:
- **Nombre**: Identificador único
- **Tipo**: Define el comportamiento del elemento
- **Valores**: Parámetros específicos según el tipo
- **Estilo**: Apariencia visual (color, tamaño, etc.)
