# Ayuda Rápida - Atajos de Teclado

## Archivo
- **Ctrl+N**: Nuevo proyecto
- **Ctrl+O**: Cargar proyecto
- **Ctrl+S**: Guardar proyecto
- **Ctrl+E**: Exportar como imagen
## Edición
- **Ctrl+Z**: Deshacer
- **Ctrl+Y**: Rehacer
- **Delete**: Eliminar selección
- **Ctrl+A**: Seleccionar todo
## Vista
- **Ctrl++**: Zoom in
- **Ctrl+-**: Zoom out
- **Ctrl+0**: Reset zoom
## General
- **F1**: Ayuda rápida
- **ESC**: Cancelar operación
- **Space**: Mover vista
## Navegación
- **Flechas**: Mover selección
- **Ctrl+Flechas**: Mover rápidamente
- **Tab**: Siguiente elemento
- **Shift+Tab**: Elemento anterior
