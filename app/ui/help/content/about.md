# Acerca de Asteroid

## Descripción
Asteroid es un editor visual de flujos de trabajo desarrollado con Python y PyQt6.
![](images/main_interface_example1.png)

## Características
- ✅ Editor de nodos visual
- ✅ Conexiones dinámicas
- ✅ Propiedades configurables
- ✅ Exportación a múltiples formatos
- ✅ Interfaz intuitiva y personalizable

## Tecnologías
- **Lenguaje**: Python 3.x
- **GUI**: PyQt6
- **Formato de proyecto**: .astr (JSON personalizado)
- **Renderizado**: QPainter
